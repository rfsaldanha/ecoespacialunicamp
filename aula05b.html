Empilhar dados

Para empilhar os dados de modo automtico, vamos precisar que a base de dados tenha uma organizao bsica.

  • O primeiro campo deve ser o de identificao;
  • Os nomes das variveis devem conter a especificao da varivel e o ano, algo como PIB2000 e PIB2010.
  • No devem existir outras variveis alm da identificao e dados do painel.

Vejamos a base de dados do exemplo.

Para colocar os dados em painel, criamos uma funo. Veja abaixo.

painel <- function(id, dados){
  require(reshape2)
  
  dadosp <- reshape2::melt(dados, id=id)
  dadosp$varname <- as.character(gsub("[[:digit:]]", "", dadosp$variable))
  dadosp$year <-  as.character(gsub("[[:alpha:]]", "", dadosp$variable))
  
  sp <- split(dadosp, f = dadosp$varname)
  
  dadosp <- data.frame(sp[[1]][,1], sp[[1]]$year)
  
  for(i in 1:length(sp)){
    dadosp <- cbind(dadosp, sp[[i]]$value)
  }
  
  names(dadosp) <- c("id","ano",names(sp)) 
  
  return(dadosp)
}
